# Hyper-Casual Games

A simple hyper-casual games app built with Flutter.

## Features
- **Game Hub**: A central menu to select games.
- **Reaction Tap**: A fast-paced tap game where players must tap the moving target as many times as possible within a 30-second time limit.

## Tech Stack
- Flutter
- Dart

## Getting Started
1. Ensure you have Flutter installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to start the app.
